# Extension Icons

This directory should contain PNG icons for the Chrome extension:

- `icon16.png` — 16×16px (toolbar)
- `icon48.png` — 48×48px (extensions page)
- `icon128.png` — 128×128px (Chrome Web Store / install dialog)

**Design suggestion**: A simple camera/document icon with "s2p" text, using the brand colors (#00d4ff on dark background).

For development with unpacked extensions, Chrome will use a default icon if these are missing.
